# swift_gyb.vim
Convert XXX.swift.gyb to XXX.swift

![GYB Sample](https://raw.githubusercontent.com/tokorom/swift_gyb.vim/assets/sample.png)

## Usage

```vim
:GYB
```

## Install

```vim
NeoBundle 'tokorom/swift_gyb.vim'
```

## Setting (Optional)

```vim
let g:swift_gyb_bin = '~/bin/gyb' "default is `gyb`
```
